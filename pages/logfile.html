Network Scanner

Python

This Python program is a network scanner that allows a user to choose between two scanning modes: ICMP (Internet Control Message Protocol) or TCP (Transmission Control Protocol). The program is designed to identify active hosts or open ports within a specified IP address range.

In the ICMP mode, the program sends ICMP echo requests (commonly known as pings) to a range of IP addresses. In the TCP mode, the program scans a specific IP address for open ports.

Overall, this tool helps users identify active hosts or open ports on a network, making it useful for network diagnostics and security assessments.
