GIF Generator

JavaScript

This web application uses the Giphy API to generate a random gif each time the page loads.

It does this by sending a request to the API, and storing the API's response as JSON data.

The URL of the GIF image from the JSON data is used to create an HTML image element, and appends this to the web page.
